# Backend node.js

Backend with graphql & express for project sharexam v2

Backend: node.js, express, apollo server, graphql, mongoose and firebase.

Database: mongoDb.

Test: jest, supertest and mongodb-memory-server.

## Develop

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## Unitary test

```bash
npm run test:unit
```

## Integration test

```bash
npm run test:int
```

## All tests

```bash
npm run test
```
